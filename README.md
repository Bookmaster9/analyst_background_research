## Analyst Dashboard

An interactive web application for exploring financial analyst insights, stock predictions, and earnings call commentary.

### Features

**Search & Discovery**

- Search the database for financial analysts
- Random example analysts with refresh functionality

**Analyst Profiles**

- View analyst information with LinkedIn information
- Browse historical stock price predictions
- Click on any price prediction to view interactive charts showing price history

**Earnings Call Commentary**

- Explore analyst commentary from earnings calls
- AI-powered chatbot for insights and analysis
- Click on any earnings call question to ask questions about analyst observations and get context-aware responses

**Tech Stack**

- Next.js 15 with TypeScript
- Tailwind CSS with custom animations
- Supabase (PostgreSQL database)
- TradingView Widget API
- OpenAI API for AI chat functionality
